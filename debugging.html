Debugging in R

Squash bugs fast

Learn to Debug R code

Debugging code is one of the most important skills after you start writing your own functions. Developing the skill of debugging takes time but here is a good starting place. Some recommended reading in Hadley Wickham's Advanced R book

Debugging Functions

  • browser()

    Put this in the body of your function, and it will pause its execution allowing you to interactively step through your function

  • debug()

    Stops a function from executing automatically and allows you to execute one statement at a time

  • print("")

    Print statements are useful to pin down where you are during a function's execution

  • VScode and RStudio Debugger

    They have some really nice interactive debugging tools that are worth exploring
