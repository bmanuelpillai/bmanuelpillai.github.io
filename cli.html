Command Line Tools

Get Fast with CLIs

Why Learn to Use CLI tools

After spending sometime programming you'll want to get faster at general routine tasks and you'll want to be able to easy explain what you did to someone else. I think graphical user interfaces are great when you are first learning an application, which is why I started my Version Control article with GitHub Desktop, but if you get comfortable with command line tools you can get a lot faster at your work and easy have someone reproduce what you did without having to point and click.
