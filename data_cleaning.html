Cleaning and Visualizing Data

The best place to start working with data is to clean and visualize that data. You should calculate descriptive statistics on your dataset before you run any statistical tests.

Tidyverse

I think learning the Tidyverse is a great way to write readable code for manipulating code and works well with ggplot to visualize that data. I recommend starting with this course and this associated Youtube series by Kelsey Codes. The course does a great job quickly getting you familiar with tidyverse and ggplot and provides associated homeworks.

Frequently Used Functions to Start

  • select()
    • Used to select specific columns from a dataset
  • filter()
    • Used to filter rows based on specific conditions
  • pivot_longer()
    • Transforms wide data into long format
  • pivot_wider()
    • transforms long data into wide format
  • ggplot()
    • Use this to start plotting and add + to add layers
