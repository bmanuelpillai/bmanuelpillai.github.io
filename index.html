Why Coding is Fun

Coming from a STEM background one of my favorite parts about programming is how you get instant feedback from running your code. Either your code runs or you get an error message. Everytime your code runs you get a feeling of accomplishment that drives you over come the next bug in your code. Everytime you squash a bug in your code you develop your skillset in diagnosing new the next bug.

I think coding is especially fun for scientists because it isn't our main job. We use code as a tool to answer the questions that interest us
