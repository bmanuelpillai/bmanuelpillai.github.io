Programming and Data Science Learning Path

Four Phases Building
Foundational Programming Skills

1

Phase 1: Getting Started

Get comfortable working with data. You should be able to manipulate a data frame easily such as filter rows, select columns, create new columns, visualize your data, and calculate basic statistics.

The next big milestone would be to get comfortable working with functions. Using functions will revolutionize how you program. A first, it may seem like writing functions takes way more time but in the long-term functions make your code more readable, less prone to bugs, and easier for you to comeback to.

  • Start with this Statistical Programming Course
    • Complete the homework assignments
  • Read R for Data Science
  • Experiment with functions, loops, and control flow in your code
  • Learn GitHub for version control
  • If machine learning is needed for your research:
    • Learn basics of Pandas and NumPy in Python
    • Use Poetry to manage Python dependencies
    • Complete ML tutorials on Kaggle
    • Read through Sci-KitLearn Documentation
    • Participate in Kaggle learning competitions
    • Study how experts approach regression and classification tasks
  • Consider learning SQL to efficiently manage and query large datasets
Note: Getting comfortable working with data is probably the hardest part. You will start out being uncomfortable with the syntax, having to constantly look things up, and getting frustrated. Remember this is the hardest part and you will be gaining a lot of momentum going forwards.
2

Phase 2: Advancing your skills

  • Read Hadley Wickham's Advanced R and R Packages
    • Work towards creating well-documented Functions to share with others
  • Learn about non-standard evaluation for using tidyverse syntax in functions
  • Develop debugging skills for functions
  • Get comfortable with:
    • Git branches, rebasing, and pull requests
    • Command-line tools and terminal navigation
    • Simple bash scripting
  • (Optional) Read "Grokking Algorithms" to understand basic data structures and algorithms
  • Focus on refining your statistics skills simultaneously
Note: This is really useful if you share code with others and do a lot of data cleaning, or other repetitive coding tasks
3

Phase 3: Apply your skills to research

  • Balance coding skills with research questions and statistical knowledge
  • Explore Bayesian frameworks that require computational power
  • Learn to use:
    • High-performance computing clusters (if available)
    • AWS or Azure cloud computing
Note: These 3 phases are the most important to have a strong foundation to data science. The next phase is a collection of things that I found useful in my own research.
4

Phase 4: Optimization and Customization

  • Switch to VS Code as your editor
  • Experiment with Vim motions
  • Customize your terminal environment
  • Watch educational videos about interesting projects
  • Participate in Kaggle projects and study other people's solutions
  • For advanced software development:
    • Learn Docker containers (like a box to package everything you need to run a piece of software)
    • Master cloud computing services (AWS or Azure)
  • Try Advent of Code if you want to challenge yourself or learn a new language
